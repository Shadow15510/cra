CRA Documentation
=================

Description
-----------
This small Python package allows to simply draw upper air sounding on a Skew-T diagramm using ``metpy`` library.

Installation
------------
This package is `available on PyPi <https://pypi.org/project/cra/>`_: ``pip install cra``.
